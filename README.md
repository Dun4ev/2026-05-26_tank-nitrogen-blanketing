# Азотная подушка в резервуарах (Tank Nitrogen Blanketing)

> **Имя репозитория на GitHub:** `tank-nitrogen-blanketing`

Репозиторий содержит материалы исследования, технико-экономического обоснования (ТЭО) и расчетов систем азотного дыхания (азотной подушки / tank blanketing) для резервуаров хранения нефтепродуктов (бензин, сырая нефть, керосин, дизельное топливо).

---

## 📌 Суть проекта

Основная цель проекта — глубокий технический и экономический анализ применимости азотной подушки в резервуарах с фиксированной крышей. Исследование отвечает на ключевой вопрос: **насколько эффективно азотирование снижает потери от испарения углеводородов и защищает резервуарный парк**.

Проект базируется на анализе международных стандартов и методик:
*   **Вентиляция и безопасность**: API 2000, ISO 28300
*   **Проектирование резервуаров**: API 650, API 620
*   **Расчет потерь от испарения**: EPA AP-42 (раздел 7.1)
*   **Физико-химические свойства**: ASTM D6377 (для сырой нефти), ASTM D5191

---

## 🔍 Основные разделы исследования

1.  **Физика процесса и мифы**:
    *   Анализ механизмов потерь: малые дыхания (*standing losses*) и большие дыхания (*working losses*).
    *   Почему замена воздуха на азот в атмосферном резервуаре сама по себе не прекращает испарение, и почему важна закрытая или подпорная система.
2.  **Сравнение продуктов**:
    *   **Бензин**: максимальный экономический и экологический эффект благодаря высокой летучести.
    *   **Нефть**: необходимость индивидуального расчета по фактическому давлению насыщенных паров (TVP).
    *   **Керосин и Дизель**: низкая летучесть делает проект малоэффективным с точки зрения окупаемости «спасенного продукта». На первый план здесь выходят вопросы безопасности и сохранения качества (защита от влаги и окисления).
3.  **Давление и конструкция**:
    *   Разбор критической ошибки «держать в атмосферном резервуаре давление в 1 бар».
    *   Работа в миллибарах (от 1 до 20 мбар / дюймы водяного столба) в соответствии с ограничениями API 650 (до 17.2 кПа).
4.  **Альтернативы и экономика**:
    *   Сравнение эффективности азотной подушки с внутренними плавающими крышами (эффективность 60–99%) и системами улавливания паров (VRU) (эффективность 90–98%).
    *   Влияние уставок регуляторов давления (Emerson) на расход азота.
5.  **Риски**:
    *   Анализ рисков удушья персонала (OSHA, CSB).
    *   Опасность разрушения резервуара при отказе дыхательных клапанов и регуляторов давления.
